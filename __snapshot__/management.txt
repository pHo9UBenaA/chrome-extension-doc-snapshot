Description
The chrome.management API provides ways to manage installed apps and extensions.

Permissions
management

You must declare the "management" permission in the extension manifest to use the management
API. For example:
{
  "name": "My extension",
  ...
  "permissions": [
    "management"
  ],
  ...
}





management.getPermissionWarningsByManifest(), management.uninstallSelf(), and
management.getSelf() do not require the management permission.
